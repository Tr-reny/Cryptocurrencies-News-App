* Android Studio
* Android Gradle Plugin Version 7.2.1
* Gradle 7.3.3
* Compile SDK version 32

## Features :

* Light and Dark Mode
* Bottom Navigation View
* Navigation Drawer
* SearchBar
* Featured News
* News Layout
